Product Brief

제품 브리핑

촬영 전 3분만 투자해 핵심 포지셔닝과 정확하게 전달해야 할 내용을 파악해 주세요.

기능 추천 등급 활용 시나리오 & 핵심 포인트
웹 이중 언어 번역 ⭐ 핵심 영어 블로그·SaaS·기술 문서 읽기. 원문+번역문 나란히 표시 — 브라우저 기본 번역보다 훨씬 자연스럽고 레이아웃 유지.
전역 입력창 번역 ⭐ 핵심 Gmail·Slack·Discord·WhatsApp 등 모든 입력창. 한국어로 입력 후 단축키 한 번으로 영어 전송 — 번역기 따로 열 필요 없음.
영상 자막 번역 ⭐ 핵심 YouTube 등 영상에 실시간 이중 언어 자막. 자막 Notion 내보내기 지원. YouTube 라이브 번역 가능 (자체 자막 방송 한정).
BabelDOC
복잡한 PDF		 ⭐ 문서 핵심 계약서·백서·보고서 번역 시 표·수식·다단 레이아웃 유지. 오픈소스 엔진 기반, GitHub Trending 기록. 자체 호스팅 지원.
AI Reply ◎ 심화 Gmail·GitHub에서 긴 내용 요약 및 영어 답장 초안 자동 생성.
회의 번역 ◎ 심화 Zoom·Teams·Google Meet 실시간 이중 언어 자막 번역.
호버·하이라이트 번역 ○ 선택 마우스 오버 또는 문장 선택 시 번역 표시. 정독·기술 문서에 유용.
이미지·스크린샷 번역 ○ 선택 복사 불가 영어 이미지·UI 스크린샷·차트 텍스트 즉시 번역.
일반 PDF·ePub ○ 선택 단순 레이아웃 전자책·매뉴얼 번역 후 이중 언어 파일 생성. 복잡한 경우 BabelDOC 사용.
✅ 영상에서 언급하면 좋은 내용
  • 「기본 기능은 완전 무료」임을 명확히 언급
  • 이중 언어 대조 효과 시연 (가장 직관적인 셀링 포인트)
  • 브라우저 기본 번역과의 비교 화면 삽입
  • Chrome 2024 연간 선정 수상 이력 언급
  • 마무리에 팔로워 전용 할인 코드 + 공식 링크 안내
⚠️ 구독자에게 안내해야 할 제한 사항
  • 일부 영상 플랫폼의 자막 번역 지원이 제한적일 수 있음
  • 복잡한 PDF 레이아웃 번역은 BabelDOC 엔진이 필요하며, 일반 모드와 결과 차이가 있음
  • 무료 버전은 월 AI 번역 글자 수 제한이 있으며, 헤비 유저는 Pro 권장
  • 모바일 앱 기능은 데스크톱 확장 프로그램과 일부 차이가 있음

Production Tips

핵심 제작 팁

전환율 높은 영상을 위한 실질적인 제안입니다. 강제 사항이 아닙니다.

🎯 콘텐츠 방향 & 오프닝 설계
1
오프닝에 긴장감을 만드세요: 실제 불편함으로 시작하세요. 예: 「아직도 Google 번역으로 영어 사이트 보세요? 이 확장 프로그램 쓰고 나서 영어 원문을 직접 읽게 됐어요」
2
실제 사용 장면을 보여주세요: 기능 인터페이스만 보여주지 말고 실제 사용 장면을 담으세요 (YouTube 시청, 영어 블로그 읽기, 이메일 작성 등).
3
비교 화면이 가장 임팩트 있습니다: 「확장 프로그램 없을 때 vs 있을 때」 비교 화면을 꼭 넣으세요. 전환율이 가장 높은 형식입니다.
4
CTA는 구체적으로: 「댓글에 공식 링크 + 할인 코드 올려뒀어요」라고 명확히 말하세요. 「검색해 보세요」만으로는 부족합니다.
📱 플랫폼 최적화 & 썸네일 설계
1
첫 프레임에 텍스트 필수 (Instagram / TikTok): 「무료」「필수 설치」같은 클릭률 높은 단어를 사용하고, 단순 인터페이스 스크린샷은 피하세요.
2
YouTube 롱폼은 챕터 추가 권장: 「설치 가이드 → 기능 시연 → 실전 데모 → CTA」 4개 챕터로 완주율을 높이세요.
3
댓글 관리 팁: 업로드 직후 첫 5개 댓글에 빠르게 답글을 달고, 공식 링크와 할인 코드를 댓글 상단에 고정하세요.
4
설치 과정을 완전하게 보여주세요: 공식 홈페이지에서 시작해 설치 단계를 건너뛰지 마세요. 구독자가 가장 막히는 구간입니다.
Privacy

개인정보 보호

번역 내용에는 민감한 정보가 포함될 수 있습니다. Immersive Translate는 개인정보 보호를 제품 설계의 핵심으로 삼고 있습니다.

🔒
프라이버시 모드 (OneAIFW)
로컬 비식별화 · 클라우드에 원문 없음
「로컬 인식 → 로컬 비식별화 → 클라우드 번역 → 로컬 복원」 순환 프로세스 적용. 이름, 주소, 계좌번호, 비밀번호, 개인키 등 민감 정보는 사용자 기기 밖으로 나가지 않습니다.
  • 민감 정보는 익명 플레이스홀더로 대체, 클라우드는 비식별화된 텍스트만 처리
  • 비식별화 데이터는 브라우저 메모리(RAM)에만 임시 저장, 브라우저 종료 시 자동 삭제
  • MIT 라이선스로 오픈소스 공개, 커뮤니티 감사 가능
🛡️
데이터 무보존
저장 안 함 · 학습 안 함 · 전구간 암호화
번역 내용 완전 무보존: 사용자가 번역한 텍스트, 웹페이지, 문서를 저장하지 않습니다. 번역 요청 및 AI 결과는 모델 학습에 사용되지 않습니다.
  • 모든 번역 요청은 암호화 전송
  • 익명 통계 데이터만 수집 (사용 빈도, 기기 모델, IP 지역)
  • 브라우저 설정으로 Google Analytics 등 추적 도구 관리 가능
📋
보안 사고 & 개인정보 문의
법규 준수 처리 · 1개월 내 응답
「정보보안 사고 관리 절차」 및 「비상 대응 관리 제도」를 수립하여, 데이터 유출 등 사고 발생 시 법규에 따라 조치하고 감독기관 및 사용자에게 통지합니다.
  • 개인정보 관련 문의: support@immersivetranslate.com
  • 1개월 이내 처리 및 답변

Quick Start

설치 / 시작 가이드

촬영 전에 직접 한 번 설치해 보세요. 4단계면 끝납니다. 제품 경험도 직접 느껴보실 수 있어요.

1
확장 프로그램 스토어 열기
Chrome / Edge / Firefox / Safari에서 확장 프로그램 스토어에서 「Immersive Translate」 검색
2
설치 클릭
「Chrome에 추가」 클릭 후 설치 완료를 기다리면 우측 상단에 확장 프로그램 아이콘이 나타납니다
3
안내에 따라 설정
최초 설치 시 사용 가이드가 나타납니다. 번역 엔진 선택 (기본 무료 엔진으로 충분), 단계별로 따라가면 완료
4
영어 페이지에서 테스트
아무 영어 사이트나 열고 확장 프로그램 아이콘을 클릭하거나 단축키를 누르면 이중 언어 대조가 바로 나타납니다
💡 촬영 팁: 공식 링크로 직접 구독자를 안내하면 직접 검색보다 전환율이 훨씬 높아집니다. 링크: immersivetranslate.com/ko/
1
앱 다운로드
iOS는 App Store에서 「Immersive Translate」 검색, Android는 Google Play 또는 공식 사이트에서 다운로드
2
권한 허용
「전역 키보드」 권한을 허용하면 모든 앱의 입력창에서 번역 기능을 사용할 수 있습니다
3
번역 엔진 선택
기본 무료 엔진으로 바로 사용 가능. 원하시면 자신의 API Key를 연결해 GPT-4 등 모델 사용 가능
4
앱에서 테스트
WhatsApp / Telegram 열고 입력창에 한국어 입력 후 번역 버튼 누르면 즉시 영어로 변환됩니다

협업 프로세스

첫 연락부터 마무리까지, 전체 프로세스를 최대한 간소화했습니다. 시간을 많이 빼앗지 않겠습니다.

01
첫 연락
협업 의향 확인
좌측 「협업 문의하기」 버튼으로 담당자에게 이메일 연락 주세요. 활동 플랫폼, 채널 링크, 구독자 수를 알려주시면 빠르게 검토합니다.
📧 이메일로 담당자 연락 📊 플랫폼 및 구독자 수 안내 🔗 채널 링크 전송
02
협의 확정
콘텐츠 방향, 보수, 제휴 프로그램 확인
콘텐츠 방향, 업로드 일정, 보수를 확인합니다. 이 페이지를 읽으셨다면 대부분의 방향은 이미 준비된 셈입니다.
또한 이 단계에서 Immersive Translate 제휴 프로그램 참여 여부를 확인합니다. 제휴 프로그램에 가입하면 전용 추천 링크를 통해 지속적인 커미션 수익을 얻을 수 있어 장기 협업을 원하는 크리에이터에게 적합합니다.
📝 콘텐츠 방향 확정 📅 업로드 일정 확정 💰 보수 확정 🔗 제휴 프로그램 확인
🔗 제휴 프로그램 자세히 보기 ↗
03
계약 및 선금
계약 체결, 보수 일부 선지급
협업 계약 체결 후 보수 일부를 선지급합니다. 계약서에 콘텐츠 요건·일정·보수 방식이 명시되어 양측의 권익을 보장합니다.
📄 계약 체결 💳 선금 지급 🔒 권익 보장
04
제작 및 검토
자유롭게 제작, 두 번의 가벼운 체크만
스타일과 언어는 자유입니다. 가벼운 검토가 두 번 있습니다: ① 스크립트 검토 (촬영 전 핵심 정보 확인) · ② 영상 검토 (업로드 전 오해 방지 확인, 스타일에는 관여 안 함).
🎨 자유 제작, 스타일 자유 📝 스크립트 검토 (촬영 전) 🎬 영상 검토 (업로드 전)
05
업로드 및 정산
영상 업로드, 잔금 지급
예정대로 업로드 후 공식 링크와 할인 코드를 댓글 상단에 고정하고 영상 링크를 전달해 주세요. 확인 후 잔금을 지급합니다.
🎁 구독자 혜택 · 주간 이용권 추첨
무료 주간 이용권 × 최소 5장
댓글란 또는 영상 마무리에서 추첨 이벤트를 진행할 수 있도록 최소 5장의 무료 주간 이용권을 제공합니다. 구독자 참여도와 전환율을 높이는 데 효과적입니다.
🏷️ 팔로워 전용 할인
Pro 전용 10% 할인 코드
팔로워가 전용 할인 코드로 Pro를 구매하면 10% 할인 혜택을 받습니다. 이 코드는 댓글 상단 고정 또는 영상 마무리에 반드시 언급해야 하며, 전환율에 직접적인 영향을 미칩니다.
🚀 예정대로 영상 업로드 🔗 댓글 상단에 링크 & 할인 코드 고정 🎁 추첨 이용권 준비 완료 💰 잔금 지급
06
특별 혜택
업로드 후 1주일, 무료 프로모션 지원 🎁
업로드 후 1주일간 성과를 모니터링합니다. 데이터가 좋으면 무료 프로모션 지원을 제공해 더 많은 잠재 구독자에게 도달할 수 있도록 돕겠습니다.

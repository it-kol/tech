Product Brief

产品简报

录制之前，请先花 3 分钟了解这个工具的核心定位、适用场景与必须准确传达的信息。

功能模块 推荐级别 典型使用场景（达人 & 粉丝） 达人推荐话术 / 核心卖点（一句话）
网页双语翻译 ★ 必讲 看英文博客、SaaS 后台、文档、新闻。 原文+译文并排，比浏览器机翻更适合深度阅读，也更方便自己校对内容。
全局输入框翻译 ★ 必讲 在 Gmail、Slack、Discord、WhatsApp 等任何输入框，用中文打字一键变英文。 在各种聊天和邮筱里用中文思考，一键变地道英文发出去，不用再来回复制到翻译器。
视频字幕翻译 ★ 必讲 YouTube 教程、评测、会议录像实时双语字幕，可导出。支持 YouTube 直播实时翻译（仅限自带字幕的直播）。 看英文视频直接加双语字幕，还能一键导出字幕丢进 Notion 做学习笔记。
BabelDOC（复杂 PDF 翻译） ★ 文档必讲 翻译合同、技术白皮书、复杂报告，保持表格、公式、多栏排版不乱。 复杂 PDF 翻译后排版不崩，底层是开源引擎，还上过 GitHub 热门榜，适合在意专业度和隐私的团队。
AI Reply ◎ 进阶可讲 在 Gmail / GitHub 中一键总结长内容，生成英文回复草稿。 长篇英文邮件一键总结再起草回复，你只需要微调语气就能发。
会议翻译 ◎ 进阶可讲 Zoom / Teams / Google Meet 会议实时字幕翻译。 开全球远程会议时自动出双语字幕和聊天翻译，不再“开会空耳”。
划词 / 悬停翻译 ○ 可选 / 进阶 精读长文、技术文档时，对某个词或段落做快速查询。 精读时划一下或鼠标一停就出翻译，比整页机翻更精细、打扰更少。
图片 / 截图翻译 ○ 可选 / 进阶 翻译软件界面截图、图表里的英文、产品包装图等。 遇到只能截图不能复制的英文，丢进来就能翻译，很适合临时救急。
普通 PDF / ePub 翻译 ○ 可选 / 进阶 快速翻译排版简单的电子书、手册或报告，生成双语文件。 文档不复杂就用普通 PDF 翻译，一键生成双语版；复杂的再切到 BabelDOC 模式即可。
✅ 建议在视频中提到
  • 明确说明「基础功能完全免费」
  • 展示双语对照效果（最直观的卖点）
  • 与浏览器自带翻译做对比镜头
  • 提到 Chrome 2024 年度精选这个背书
  • 结尾说明粉丝专属折扣码 + Landing Page 链接
⚠️ 需要向观众说明的限制
  • 部分视频平台（如爱奇艺）字幕翻译支持有限
  • 复杂 PDF 排版翻译需要使用 BabelDOC 引擎，普通模式效果有差异
  • 免费版每月有 AI 翻译字数限制，重度用户建议 Pro
  • 移动端 App 功能与桌面插件略有差异

Production Tips

关键制作要点

这些是帮助你做出高转化视频的实质性建议，不是强制要求。

🎯 内容定位与开场设计
1
开场必须有冲突感：用真实痛点开场，不要用“大家好我是某某”。例：「你还在用谷歌翻译看英文网页？这个插件让我开始看英文原文」。
2
展示真实使用场景：不要只展示功能界面，要展示你真实在用的场景（看 YouTube 教程、读英文博客、写英文邮件等）。
3
对比镜头最有冲击力：如果有条件，加一个“不用插件 vs 用了插件”的对比镜头，这是转化率最高的内容形式。
4
CTA 必须具体：结尾说清楚「在评论区放了 Landing Page 链接 + 粉丝专属折扣码」，不要只说“去搜一搜”。
📱 平台适配与封面设计
1
抖音 / 小红书首帧必须有文字：封面文字建议用“免费”“必装”“我用了 X 年”等高点击词，避免纯展示界面截图。
2
B 站长视频建议加分章标题：分章标题能显著提升完播率，建议分“安装教程”“功能展示”“实战演示”“CTA”四个章节。
3
评论区运营小技巧：发布后第一时间回复前 5 条评论，并主动把 Landing Page 链接和折扣码置顶，能显著提升转化。
4
展示安装流程要完整：展示安装时建议从官方 Landing Page 进入安装，不要直接跳过安装步骤，这是观众最容易卡壳的环节。
Content Ideas

内容灵感 / 脚本建议

4 种创作角度，每种都有完整的开场、展示和 CTA 话术参考。风格是你的，方向我们帮你想好了。

⚠️
功能介绍不要超过 3 个（B 站长视频达人除外）—— 短视频和中长视频功能介绍越多，观众记忆越模糊，转化率反而越低。建议选 1–2 个与你粉丝最相关的功能深度展示，而不是把所有功能走马观花。 B 站长视频（超过 8 分钟）可适当展开到 3–5 个功能，但每个功能建议至少展示 60 秒以上。
😤 痛点型
「看英文内容，你还在用浏览器自带翻译？」
适合短视频 / 竖屏 · 60–90 秒 · 适合小红书、抖音、B站短片
开场钩子（前 5 秒）
「你有没有遇到过这种情况——打开一个英文网站，用浏览器翻译一键翻，结果整个页面乱成一锅粥，排版全崩了？」
中段展示（重点演示 2 个功能）
① 对比镜头：Chrome 自带翻译 vs 沉浸式翻译双语对照，视觉冲击最强
② 顺手带一下全局输入框翻译：在 Gmail 里直接中译英发邮件，不用开 ChatGPT
结尾 CTA
「完全免费，Chrome 商店直接搜'沉浸式翻译'就能装，链接放评论区了，还有粉丝专属 9 折折扣码和周卡抽奖，评论区见！」

② 顺带一提：视频字幕双语——YouTube 外语教程开双语字幕，一句话带过即可

Quick Start

安装 / 上手指引

录制前先自己装一遍，4 步搞定，顺便感受一下产品体验。

1
打开插件商店
Chrome / Edge / Firefox / Safari，在扩展商店搜索「沉浸式翻译」
2
点击安装
点击「添加到 Chrome」，等待安装完成，右上角出现插件图标
3
跟着引导走
首次安装会有使用引导，选择翻译引擎（默认免费引擎即可），一步步走完
4
打开英文网页试试
打开任意英文网站，点击插件图标或按快捷键，双语对照立刻出现
💡 录制建议：可以用 Landing Page 链接直接引导观众安装，比让他们自己搜索转化率更高。链接：immersivetranslate.com/zh-Hans/
1
下载 App
iOS 在 App Store 搜索「沉浸式翻译」，Android 可在官网下载 APK
2
开启权限
开启「全局输入法」权限，即可在任意 App 的输入框中使用翻译功能
3
选择翻译引擎
默认免费引擎即可使用，也可接入自己的 API Key 使用 GPT-4 等模型
4
在 App 里试试
打开 WhatsApp / Telegram，在输入框里打中文，点击翻译按钮，中译英立刻完成

创作与审核
自由创作，两个节点对齐一下就行
我们鼓励你用自己的风格和语言创作，不需要按照脚本字字照念。有两个轻量审核节点：①脚本审核——开拍前我们确认一下关键信息有没有跑偏；②视频审核——发布前快速过一遍，主要是避免误导观众，不干涉你的内容风格。
